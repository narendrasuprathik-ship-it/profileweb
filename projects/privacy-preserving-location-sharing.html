Live Bangalore Map Sandbox

  • Designed a geo-indistinguishability engine using Planar Laplace noise.
  • Added configurable ε and blur radius to protect GPS signals.
  • Focused on real-world location-based service (LBS) privacy risks.
  • Built a Streamlit + Folium dashboard with a Bangalore live demo.
  • Implemented privacy controls and exportable noisy datasets.
  • Generated privacy/utility metrics for downstream analysis.
  • Quantified trade-offs using mean error and service accuracy.
  • Simulated adversarial attacks (home, trajectory, clustering).
  • Showed lower ε boosts privacy while degrading LBS accuracy.
Key insight

Privacy budgets directly shape utility; ε tuning must match the risk profile of each LBS scenario.

Next enhancement

Integrate adaptive ε selection based on context sensitivity and user intent.
